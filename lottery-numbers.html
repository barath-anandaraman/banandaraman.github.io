Lottery Numbers Analysis

Problem Statement

Are Lottery Numbers winning sequences really random?

Data Used

The dataset for this project was obtained from public dataset. Key datasets includes:

  • Mega Millions
  • Powerball
  • New York Pick 10

Methodology

The project followed the following steps:

  1. Data Cleaning & Preprocessing:
    • Use Python for preprocessing
    • Merge datasets needed for Tableau dashboard creation
  2. Tableau dashboard:
    • Histogram for Frequency of Drawn Numbers
    • Smoothed Frequency of Numbers using Line Chart
    • Bubble chart for Top repeated numbers
    • Density Plot for concentration of Drawn numbers
    • Small multiple Number frequency chart by Drawn position
    • Heatmap of Lottery number patters over years

Tools & Technologies

  • Languages: Python, Tableau
  • Libraries: Pandas,
  • Development Environment: Jupyter Notebooks, Tableau
  • Version Control: Git, GitHub
