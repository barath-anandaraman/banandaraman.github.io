Movie Recommender System

Problem Statement

Create a Movie Recommender system that is Content based and collaborative filtering.

Data Used

The dataset for this project was obtained from Movie Lens database (http://grouplens.org/datasets/). Key datasets included:

  • Movies
  • Ratings
  • Tags
  • Links

Methodology

The project followed a standard data science lifecycle:

  1. Content based Recommender:
    • Create Movie Content Genre + tag
    • Vectorize using TfidfVectorizer and Apply Nearest Neighbors.
    • Use MovieId for indexing, to perform search by movieID based on title, and Build mappings
    • Create Recommender Function
    • Test with Movie
  2. Collaborative Filtering - Based Recommender
    • Prepare user-Movie Ratings matrix for collaborative filtering.
    • Matrix Factorization using Truncated SVD and fit nearest neighbors on latent factors.
    • Use MovieId for indexing, to perform search by movieID based on title, and Build mapping
    • Build Recommendation Function for collaborative filtering
    • Test with Movie

Tools & Technologies

  • Languages: Python
  • Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, TfidfVectorizer, NearestNeighbors
  • Development Environment: Jupyter Notebooks
  • Version Control: Git, GitHub
